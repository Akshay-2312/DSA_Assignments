# Complexity Analysis

## Graph Operations

- Add Node: O(1) amortized
- Add Edge (list + matrix): O(1)
- Display List: O(V + E)
- Display Matrix: O(V²)

## Dijkstra (single-source)

- Time: O(E log V) with priority queue
- Space: O(V + E)

## Floyd–Warshall (all-pairs)

- Time: O(V³)
- Space: O(V²)

## Trees

- Binary Tree Insert/Search: O(h), worst O(n)
- AVL Insert/Search: O(log n)
- Traversals: O(n)

## Topological Sort (Kahn)

- Time: O(V + E)
- Space: O(V)

## Prim MST

- Time: O(E log V)
- Space: O(V + E)
